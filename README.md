Budget Ops Dashboard

Excel-based budget operations analysis dashboard with QA and risk detection features.

Overview

This project aims to convert an Excel-based operational budget workflow into a lightweight analysis dashboard.

The app will analyze budget progress, detect risk conditions, and display operational warnings based on uploaded Excel files.

Planned Features

- Excel upload
- Monthly budget analysis
- Budget consumption rate
- Risk detection
- Red / Yellow / Green status judgment
- Automated warning comments
- Vendor-based cost analysis
- Simple dashboard UI

Goal

The goal is to support frontline operational budget management with clearer visualization and QA-style analysis.

Status

Currently in planning and early prototype phase.

Planned Analysis Logic

The dashboard is planned to evaluate operational budget conditions using Excel-based monthly data.

Example status logic:

- Green:
  Budget usage is within the expected range.

- Yellow:
  Budget consumption pace is slightly higher than expected.

- Red:
  Monthly forecast exceeds the planned budget.

The system will also generate warning comments to support early operational decision-making.

Future Ideas

- Budget anomaly detection
- Monthly comparison dashboard
- Excel automation support
